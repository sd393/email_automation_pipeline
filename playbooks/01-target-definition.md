# Purpose

Guidelines for the Stage 0 brief interview: how to ask each question, what defaults to suggest, how to handle vague user answers (e.g., "all retailers" → ask for revenue bands), and how to confirm the segment definition before writing `brief.yaml`.

# When Claude reads this

When filling the `target.*`, `who_to_contact.*`, and `message.*` sections of the brief during Stage 0, especially if the user's one-sentence ask is ambiguous.
